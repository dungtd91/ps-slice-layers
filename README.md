<p>This photoshop action creates a slice from multiple selected layers. The created slice will occupy the area of all layers combined.</p>

<p>Select all the layers from which you wish to create a slice and then run the action. You may run the action from the Photoshop Actions Panel (Window&gt;Actions) or by using the cmd+shift+F2 shortcut.</p>

<p><strong>Please note you need to select more than one Photoshop layer or a folder for the action to work.</strong></p>
